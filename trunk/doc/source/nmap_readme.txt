nMap Package
============

Yet another terrain implementation for Nebula.  Uses Willem H. de Boer's
geometric mipmapping technique with a static vertex buffer to take advantage
of modern graphics cards.

Author: Ling Lo
Version: 1.03
License: Same as Nebula, see nebula/doc/source/nmap_license.txt

Running demo script
===================

To run the test script, use the following command in nebula/data/nmap:

    nsh -run startup.tcl

Controls:

   mouse - same as tekdemos
   f5    - shaded terrain
   f6    - wireframe terrain
   f9    - take a screenshot, will overwrite each other
